## Contributing

Hi there! We're thrilled that you'd like to contribute to this project. Your help is essential. Our [README](README.md) describes the project, its purpose, caveats, and some ways to contribute.

Contributions to this project are released to the public domain under [CC0-1.0](LICENSE.md).

Please note that this project is released with a [Contributor Code of Conduct](CODE_OF_CONDUCT.md). By participating in this project you agree to abide by its terms.
